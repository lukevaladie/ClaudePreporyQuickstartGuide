Chat Basics

The chat interface is where you'll spend most of your time. Here's what matters.

Pick the right model

Click the model selector below the input to switch between models:

🔮
Opus 4.6
Most capable. Complex analysis and nuanced work.
⚡
Sonnet 4.6
Balanced speed and intelligence. Great for quick tasks.
🏃
Haiku 4.5
Fastest. Quick questions and simple tasks.

Search, tools & file uploads

The Search and tools button (lower-left of input) is your control center for toggling extended thinking, web search, and styles. The + button handles file uploads — PDFs, spreadsheets, images, and more (up to 30 MB each).

Tip Say "search the web for…" in your message to make sure Claude pulls live results.

Styles

Styles adjust Claude's tone and format. Four presets are built in — Normal, Concise, Formal, and Explanatory — and you can create custom ones from writing samples or descriptions.

✨

Artifacts

When Claude creates something substantial — a document, code, an interactive app — it appears in a dedicated panel to the right. These are artifacts: Claude's deliverables.

What can Claude create?

📄
Documents
Markdown, Word, PDF, PowerPoint
💻
Code & Apps
Any language, interactive React apps
📊
Visuals
Charts, SVGs, diagrams, HTML pages

Iterate and share

Ask for changes and Claude updates the artifact in place. Flip through versions with the version selector. Hit Share to create a team-only link — only authenticated Prepory members can view it.

📁

Projects

Persistent workspaces where every conversation shares the same knowledge base and instructions.

The sidebar

Your home base for navigating chats, projects, and artifacts.

Create a project

1
Click Projects in the sidebar, then + New Project.
2
Give it a name and description.
3
Choose visibility — private or shared with your org.

Add knowledge & instructions

Upload documents to the knowledge base — unlike single-chat uploads, these are available to every conversation in the project. Set project instructions to tell Claude what role to play, what format to use, or what terminology matters.

Share with teammates

Add members with Can use (chat only) or Can edit (full control) permissions.

Tip Your chats in a shared project stay private by default — everyone gets their own conversations using the shared knowledge.
🧠

Memory & Preferences

Help Claude understand who you are so every conversation is tailored to you.

Profile preferences

Open Settings (click your initials, lower-left) and fill in your name, role, and communication preferences. These apply across all conversations — set once, benefit everywhere.

Memory

Memory lets Claude learn from your conversations over time — your projects, working style, and preferences. Toggle it on in Settings → Capabilities. You can review, edit, or delete memories at any time, or simply tell Claude to remember or forget things in conversation.

Tip Use Incognito mode (ghost icon, upper-right) for conversations you don't want saved to history or memory.
🤖

Cowork Desktop App

Hand off entire tasks — not just questions, but outcomes. Claude plans, reads files, and delivers finished work.

How it works

1
Open the Claude Desktop app and click the Cowork tab.
2
Describe the outcome — "organize these files," "turn these notes into a presentation."
3
Review Claude's plan, grant folder access, and let it run.

Great for

📂
File organization
Sort, rename, and restructure
📝
Document creation
Notes → polished docs & slides
📊
Data analysis
Spreadsheets with working formulas
Tip Cowork requires the desktop app and won't run in the browser. Keep the app open while Claude works.
